📘 New Topic
Linked List — Delete Last Node

Today you'll learn your first interview-level pointer manipulation.

Observation

Delete the last node.

Before

Head
 ↓
10 → 20 → 30 → 40 → None

After

Head
 ↓
10 → 20 → 30 → None
Important Question

Do we search for 40?

No.

Exactly like yesterday,

we search for the node before it.

10 → 20 → 30 → 40
              ↑
           current

When

current.next.next == None

you know

current

is standing at the second last node.

Then

current.next = None

Done.

Visualization
10 →20→30→40→None
          ↑
      current

Change only

30.next

from

40

to

None

Now

40

becomes unreachable.

Algorithm
if list empty
    return

if only one node
    head = None

else

move until

current.next.next == None

current.next = None
Time Complexity
O(n)
Space Complexity
O(1)
