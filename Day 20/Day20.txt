Concept: Insertion Sort
Real-Life Analogy

Imagine you're holding playing cards.

You pick one new card at a time and insert it into the correct position among the cards already sorted in your hand.

Example:

Cards in hand:

2 4 5

New card:

3

Insert it:

2 3 4 5

That's exactly how Insertion Sort works.

Example

Array:

[5,2,4,1,3]
Pass 1
5 | 2 4 1 3

Insert 2 before 5

2 5 | 4 1 3
Pass 2

Insert 4

2 4 5 | 1 3
Pass 3

Insert 1

1 2 4 5 | 3
Pass 4

Insert 3

1 2 3 4 5

Done.

Algorithm
Assume the first element is already sorted.
Pick the next element (key).
Compare it with elements on the left.
Shift larger elements one position to the right.
Insert the key into its correct position.
Code
def insertion_sort(arr):

    for i in range(1, len(arr)):

        key = arr[i]
        j = i - 1

        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1

        arr[j + 1] = key

    return arr
