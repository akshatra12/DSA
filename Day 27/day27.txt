🌟 New Concept — Monotonic Arrays

This is a small but important interview concept.

What is a Monotonic Array?

An array is monotonic if it is entirely increasing or entirely decreasing.

Increasing
[1,2,3,4,5]

Every next element is:

>= previous
Decreasing
[9,7,5,3]

Every next element is:

<= previous
Also Monotonic
[1,2,2,3,3]

Because duplicates are allowed.

Not Monotonic
[1,3,2,4]

Why?

1 → 3 ↑

3 → 2 ↓

2 → 4 ↑

Direction changes.

Observation

There are only two possibilities.

Either

Increasing

or

Decreasing

So we maintain two variables.

increasing = True
decreasing = True

Traverse once.

If you ever see

arr[i] > arr[i+1]

Then

increasing = False

Similarly,

If you see

arr[i] < arr[i+1]

Then

decreasing = False

At the end,

If either is True,

the array is monotonic.

Dry Run
arr = [1,2,2,3]

Initially

Increasing = True

Decreasing = True

Compare

1 < 2

Not decreasing anymore.

Decreasing = False

Continue.

Nothing breaks increasing.

Answer:

True
