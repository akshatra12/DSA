Concept: Selection Sort

Before coding, understand the idea.

Bubble Sort

Bubble Sort:

Pushes the largest element to the end.
Selection Sort

Selection Sort:

Finds the smallest element
and puts it at the beginning.

Example:

[5,2,4,1,3]
Pass 1

Find smallest:

1

Swap with first element:

[1,2,4,5,3]
Pass 2

Remaining:

2,4,5,3

Smallest:

2

Already in correct position.

Pass 3

Remaining:

4,5,3

Smallest:

3

Swap:

[1,2,3,5,4]
Pass 4

Remaining:

5,4

Smallest:

4

Swap:

[1,2,3,4,5]

Done.

Selection Sort Algorithm
For every position i:

Find smallest element
from i to end

Swap with i
Code
def selection_sort(arr):

    n=len(arr)

    for i in range(n):

        min_idx=i

        for j in range(i+1,n):

            if arr[j] < arr[min_idx]:
                min_idx=j

        arr[i],arr[min_idx]=arr[min_idx],arr[i]

    return arr
