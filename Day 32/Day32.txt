🧠 Today's Concept: Creating a Node

Until now we've used arrays:

arr = [10,20,30]

A Linked List stores each value inside a Node.

Each node has:

data → the value
next → address of the next node

Visual:

+------+-------+
| Data | Next  |
+------+-------+
|  10  |   ●--------+
+------+-------+    |
                    v
              +------+-------+
              |  20  |   ●--------+
              +------+-------+    |
                                  v
                            +------+-------+
                            |  30  | None  |
                            +------+-------+
Step 1: Create a Node Class

Read this carefully.

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

Meaning:

When we write

node = Node(10)

Python creates

Data = 10

Next = None
Step 2: Connect Nodes

Suppose

a = Node(10)
b = Node(20)
c = Node(30)

Initially

10    20    30

None  None  None

Now connect them

a.next = b
b.next = c

Result

10 → 20 → 30 → None
Step 3: Traversing a Linked List

Suppose

Head

↓

10 → 20 → 30 → None

How do we print every value?

Think like this:

Current = Head

Print Current

Move to Next

Repeat

Until Current becomes None

This idea is the foundation of every linked list problem.

Common Mistakes
❌ Mistake 1
current = current.next.next

You skip nodes.

❌ Mistake 2

Forget to move

current = current.next

Infinite loop.

❌ Mistake 3
while current.next:

This misses the last node.

Instead think carefully about when traversal should stop.
