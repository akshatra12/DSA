🌟 New Concept: 2D Arrays (Matrices)

Until now, you've worked with 1D arrays:

arr = [10,20,30,40]

Indexing:

0 1 2 3
What is a 2D Array?

A 2D array is an array of arrays.

Example:

matrix = [
    [1,2,3],
    [4,5,6],
    [7,8,9]
]

Think of it like a table.

1 2 3
4 5 6
7 8 9

Rows:

0
1
2

Columns:

0 1 2
Accessing Elements
matrix[0][0]

Output:

1
matrix[1][2]

Output:

6
matrix[2][1]

Output:

8
Traversing a Matrix
for row in matrix:
    for value in row:
        print(value)

Output:

1
2
3
4
5
6
7
8
9
Why Are Matrices Important?

Many interview problems are based on them:

Image Rotation
Sudoku
Maze Problems
Flood Fill
Islands
Graphs

So we need a strong foundation.
