🌟 New Concept: Hash Set

You already know Dictionary (Hash Map).

Today we'll learn Hash Set.

Many beginners ask:

"Why do we need a set if we already have a dictionary?"

Good question.

List
arr = [1,2,2,3]

Duplicates are allowed.

Set
s = {1,2,2,3}

Output:

{1,2,3}

A set automatically removes duplicates.

Why Is It Useful?

Suppose I ask:

Does the number 100 exist?

Using a list:

100 in arr

takes up to O(n) time.

Using a set:

100 in s

takes about O(1) time on average.

That's why sets are powerful.

Example
arr = [1,2,3,2,5,1]

seen = set()

for num in arr:
    if num in seen:
        print("Duplicate:", num)
    else:
        seen.add(num)

Output:

Duplicate: 2
Duplicate: 1

Notice:

seen.add(num)

instead of

freq[num] = 1
