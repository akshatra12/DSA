Definition

A Stack follows the principle:

LIFO

Last In, First Out

Example:

Push 10

Stack:
10
Push 20

20
10
Push 30

30
20
10

Now remove one element.

Which one comes out?

30

Not 10.

Not 20.

Because it was inserted last.

Stack Operations
Operation	Meaning	Complexity
Push	Insert on top	O(1)
Pop	Remove from top	O(1)
Peek/Top	View top element	O(1)
isEmpty	Check if stack is empty	O(1)
Size	Number of elements	O(1)

Memorize this table. Interviewers ask it frequently.

Stack in Python

Python already provides an easy implementation using a list.

stack = []

stack.append(10)
stack.append(20)
stack.append(30)

print(stack)

Output:

[10, 20, 30]

Now:

stack.pop()

Output:

30

Remaining:

[10,20]
Why append() and pop()?

Remember arrays?

Appending at the end of a Python list is efficient.

Removing from the end is also efficient.

So Python lists naturally behave like a stack.
