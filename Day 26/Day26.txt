The Recursive Function

Today don't memorize this.

Understand it.

merge_sort(arr):

    if len(arr) <= 1:
        return arr

    mid = len(arr)//2

    left = merge_sort(left_half)

    right = merge_sort(right_half)

    return merge(left,right)

This looks exactly like recursion.

Because it is.
