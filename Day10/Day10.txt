Concept: Functions

Until now you've written code like:

a = 10
b = 20
print(a+b)

But imagine needing this 100 times.

Instead we create a function.

What is a Function?

A function is a reusable block of code.

Example:

def greet():
    print("Hello")

Call it:

greet()

Output:

Hello
Function With Parameters
def greet(name):
    print("Hello", name)

greet("Akshatra")

Output:

Hello Akshatra
Function Returning a Value
def add(a,b):
    return a+b

result = add(10,20)

print(result)

Output:

30
