Last In

Queue Operations
Operation	Meaning	Complexity
Enqueue	Insert at rear	O(1)
Dequeue	Remove from front	O(1)
Front	First element	O(1)
Rear	Last element	O(1)
isEmpty	Check empty	O(1)

Memorize these.

Python Queue

The best implementation is not a list.

Instead use:

from collections import deque

queue = deque()

Why?

Because

list.pop(0)

is

O(n)

while

deque.popleft()

is

O(1)

This difference matters in interviews.

Example
from collections import deque

queue = deque()

queue.append(10)
queue.append(20)
queue.append(30)

Queue

10 20 30

Now

queue.popleft()

removes

10

because Queue follows FIFO.
