🌟 New Concept: Kadane's Algorithm

Today you're learning your first famous interview algorithm.

This algorithm appears in:

Amazon
Microsoft

Adobe

Flipkart

interviews.

Problem

Given:

arr = [-2,1,-3,4,-1,2,1,-5,4]

Find the maximum sum subarray.

Notice:

This is NOT a fixed window.

The size is unknown.

Example:

[4,-1,2,1]

Sum:

6

That's the answer.

Why Can't We Use Sliding Window?

Sliding Window works only when the window size is fixed.

Example:

Window size = 3

But here:

The best subarray could have:

2 elements
5 elements
20 elements

We don't know.

That's why Kadane's Algorithm exists.

Intuition

Imagine carrying a bag of money.

Every positive number adds money.

Every negative number removes money.

If your bag becomes negative, would you keep carrying it?

No.

You'd throw it away and start fresh.

That's exactly what Kadane's Algorithm does.

Dry Run
[-2,1,-3,4,-1,2,1,-5,4]

Current Sum:

-2

Negative.

Throw it away.

Start from:

1

Then:

1-3=-2

Negative.

Throw it away.

Start from:

4

Continue:

4
3
5
6

Maximum:

6
Kadane's Algorithm
current_sum = 0
maximum_sum = arr[0]

for num in arr:

    current_sum += num

    maximum_sum = max(maximum_sum, current_sum)

    if current_sum < 0:
        current_sum = 0

We'll discuss why this works after you attempt the problems.
