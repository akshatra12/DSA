📘 New Topic
Linked List – Insert at End

Yesterday you inserted at the beginning.

Today we'll insert at the end.

Current List
Head

↓

10 → 20 → 30 → None

Insert

40

Expected

10 → 20 → 30 → 40 → None
Before Coding
Draw it

Current

Head

↓

10 → 20 → 30 → None

New Node

40 → None

Question

Where should

new_node.next

point?

Answer:

None

Because it becomes the last node.

Which pointer changes?

Think carefully.

We do NOT change

head

We do NOT change

current

Only one pointer changes.

Which one?

👉 The next pointer of the current last node.

Example

Before

30 → None

After

30 → 40 → None

Only

30.next

changes.

Algorithm
Create a new node.
If head == None, make the new node the head.
Otherwise:
Start from head.
Move until you reach the last node (current.next == None).
Connect the last node to the new node.
Return the updated head.
